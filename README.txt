Hello, this bot it's my first work with pyTelegramBotAPI.
Привет, это моя первая работа с pyTelegramBotAPI.
Схема работы данного телеграмбота:
    -использование базы данных Sqlite3;
    -парсинг курсов валют с сайта НБРБ;
    -использование pyTelegramBotAPI для приема/отправки сообщений пользователю.
Имеет функции:
    -просмотр курсов валют;
    -обмен валют;
    -подписка на ежедневную рассылку.
Порядок действий для начала работы с ботом:
    -создать бота с помощью @BotFather в telegram и сгенерировать TOKEN;
    -созданный TOKEN прописать в config.py;
    -создать виртуальную среду;
    -с помощью pip установить данные пакеты:
        pip install bs4==0.0.1
        pip install pyTelegramBotAPI==4.2.0
        pip install lxm==4.6.4
    -запустить bot.by и parse_script.py  для начала работы бота;
    -перейти в телеграмме в чат с ботом по @%bot_name%, который вы написали в @BotFather, прописать '/start';
    -вы великолепны!
